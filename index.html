A Console Screen usually means a keyboard and mouse pair, which allows a user to input commands and receive visual output from a computer or computer system. It can also refer to specialized computers only meant to play video games. Console is a basic computer or monitor and keyboard that is connected to another computer, server, or a mainframe over a network. It is used to maintain or monitor the status of the network or computer. The Console window displays status messages for various features in the integrated development environment (IDE). To open the Console window, on the menu bar, choose View > Console, or press Ctrl+Alt+O.
